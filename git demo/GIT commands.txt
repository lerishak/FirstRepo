GIT COMMANDS

whenever a git repo is created a default branch is created which is called master.


or create a new repository on the command line
echo "# FirstRepo" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/lerishak/FirstRepo.git
git push -u origin main

or push an existing repository from the command line
git remote add origin https://github.com/lerishak/FirstRepo.git
git branch -M main
git push -u origin main
